
# Steps
1.0 Download Arduino IDE 1.8.19 from the website."https://www.arduino.cc/en/software"
2.0 Open Arduino IDE:
3.0 Click File > Preferences
	3.1 Write the following URL in (Additional Board Manager URLs) Field Then Click ok."https://dl.espressif.com/dl/package_esp32_index.json"
4.0 Click Tools > Board > Boards Manager
	4.1 Write "ESP32" and then click install.
5.0 Plug ESP32 board to the computer USB port.
6.0 Click Tools > Board > ESP32 Arduino > WEMOS D1 MINI ESP32.
7.0 Clicl Tools > Port > COM 

# Testing
1.0 Click File > Example > 01 Basics > Blink.
2.0 Press the arrow icon to upload
The light will turn ON and OFF
